Virtual Environment in Python

Virtual environment in Python is used to create isolated project spaces. It allows: • Separate dependencies • No package conflict • Clean development setup Purpose: • Better Project Management • Safe Package Installation • Independent Environments 

python -m venv myenv

Creating Virtual Environment

Virtual environment is created using venv module. 

python -m venv env

Activating Virtual Environment

Activation is required to use environment. 

# Windows
env\Scripts\activate

# Mac / Linux
source env/bin/activate

Installing Packages

Packages are installed inside virtual environment using pip. 

pip install numpy
pip install flask

Checking Installed Packages

We can check installed packages using pip list. 

pip list

Deactivating Environment

Virtual environment can be exited anytime. 

deactivate

Why Use Virtual Environment

Virtual environment helps in managing Python projects efficiently. 

• Avoid dependency conflicts  
• Project isolation  
• Clean workspace  
• Better development control
