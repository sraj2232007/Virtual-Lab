GUI Programming in Python - VII

GUI Programming in Python allows developers to create graphical user interfaces for their applications. It is an important aspect of software development that enhances the user experience and makes applications more interactive. GUI Programming improves software development by providing a visual approach to building applications, making it easier for developers to design and implement user interfaces. This leads to better user experience and more maintainable code. In this lab, you will study different GUI Programming concepts in Python, including creating windows, adding widgets, managing layouts, and handling events.

GUI Programming in Python

Creating a Window

Window is the main screen of GUI application. Tk() creates the window. 

from tkinter import *

root = Tk()

root.title("My App")

root.geometry("400x300")

root.mainloop()

Label Widget

Label displays text inside the GUI window. 

from tkinter import *

root = Tk()

label = Label(
root,
text="Hello Python"
)

label.pack()

root.mainloop()

Button Widget

Button executes functions when clicked. 

from tkinter import *

def click():
    print("Button Clicked")

root = Tk()

btn = Button(
root,
text="Click",
command=click
)

btn.pack()

root.mainloop()

Entry Widget

Entry widget is used to take input from user. 

from tkinter import *

root = Tk()

box = Entry(root)

box.pack()

root.mainloop()

Frame Widget

Frame organizes widgets inside sections. 

from tkinter import *

root = Tk()

frame = Frame(root)

frame.pack()

root.mainloop()

Message Box

Message box displays popup notifications. 

from tkinter import *
from tkinter import messagebox

root = Tk()

messagebox.showinfo(
"Info",
"Hello"
)

root.mainloop()

Layouts control widget positions.

Event Handling

Events trigger actions inside GUI applications.
