Data Types in Python - I

Python data types are used to define the type of data that a variable can hold. In Python, there are several built-in data types such as integers (int), floats (float), strings (str), booleans (bool), lists, tuples, dictionaries, and sets. Python is dynamically typed, which means you don't need to declare the data type explicitly.

Note: Before learning about operators in Python, it's important to know how to take input from users. In Python, we use the input() function to accept user input from the keyboard.

Taking Input in Python

The input() function is used to take input from the user. It reads a line from input as a string. You can convert it to other data types using functions like int(), float(), etc.

age = int(input("Enter your age: "))

Program:

# Taking input from user
age = int(input("Enter your age: "))
name = input("Enter your name: ")

print("Name: " + name)
print("Age: " + str(age))

Output:

Enter your age: 24 (user's-choice)
Enter your name: John (user's-choice)
Name: John
Age: 24

Python Data Types

Python has the following main data types:

Integer (int)

Integers are whole numbers without any decimal point. They can be positive or negative.

  • Positive Integer : 10, 25, 100
  • Negative Integer : -5, -20, -100
  • Zero : 0

Program:

a = 10
b = -5
c = 0

print("Integer a: " + str(a))
print("Integer b: " + str(b))
print("Integer c: " + str(c))
print("Type of a: " + str(type(a)))

Output:

Integer a: 10
Integer b: -5
Integer c: 0
Type of a: <class 'int'>

Float (float)

Floats are numbers with decimal points. They represent real numbers.

  • Positive Float : 3.14, 2.5, 10.0
  • Negative Float : -5.5, -3.14
  • Scientific Notation : 1e2 (100.0), 1.5e-1 (0.15)

Program:

x = 3.14
y = -2.5
z = 1e2

print("Float x: " + str(x))
print("Float y: " + str(y))
print("Scientific notation z: " + str(z))
print("Type of x: " + str(type(x)))

Output:

Float x: 3.14
Float y: -2.5
Scientific notation z: 100.0
Type of x: <class 'float'>

String (str)

Strings are sequences of characters enclosed in single quotes ('') or double quotes (""). They are used to store text.

  • Single Quote : 'Hello World'
  • Double Quote : "Hello World"
  • Multi-line String : '''This is a multi-line string'''

Program:

str1 = 'Hello'
str2 = "World"
str3 = '''This is a
multi-line string'''

print(str1)
print(str2)
print(str3)
print("Type of str1: " + str(type(str1)))

Output:

Hello
World
This is a
multi-line string
Type of str1: <class 'str'>

Boolean (bool)

Booleans represent one of two values: True or False. They are used in conditional statements.

  • True : Represents true condition
  • False : Represents false condition
  • Comparison Result : Result of comparison operations

Program:

a = True
b = False
c = 10 > 5
d = 10 == 5

print("a: " + str(a))
print("b: " + str(b))
print("10 > 5: " + str(c))
print("10 == 5: " + str(d))
print("Type of a: " + str(type(a)))

Output:

a: True
b: False
10 > 5: True
10 == 5: False
Type of a: <class 'bool'>

Types of Operators

They are of the following types:

Arithmetic Operators

Arithmetic operators are used to perform basic mathematical operations.

  • Addition(+) : It is used to perform the addition of two numbers.
  • Subtraction(-) : It is used to perform the subtraction of two numbers.
  • Multiplication(*) : It is used to perform the multiplication of two numbers.
  • Division(/) : It is used to divide the first number by the second and return the quotient as output.
  • Modulo(%) : It is used to divide the first number by the second and return the remainder as output.
  • Floor Division(//) : It divides and returns the floor value of the quotient.
  • Exponentiation(**) : It raises the first number to the power of the second.

Program:

a = 10
b = 5

print("Addition: " + str(a + b))
print("Subtraction: " + str(a - b))
print("Multiplication: " + str(a * b))
print("Division: " + str(a / b))
print("Modulo: " + str(a % b))
print("Floor Division: " + str(a // b))
print("Exponentiation: " + str(a ** b))

Output:

Addition: 15
Subtraction: 5
Multiplication: 50
Division: 2.0
Modulo: 0
Floor Division: 2
Exponentiation: 100000

Relational Operators

Relational operators are used to compare two values.

  • Equal to(==) : It compares two values and returns True if equal, otherwise False.
  • Not equal to(!=) : It compares two values and returns False if equal, otherwise True.
  • Greater than(>) : It checks if first value is greater than second then returns True, otherwise False.
  • Less than(<) : It checks if first value is less than second then returns True, otherwise False.
  • Greater than or equal to(>=) : It checks if the first value is greater than or equal to the second then returns True, otherwise False.
  • Less than or equal to(<=) : It checks if the first value is less than or equal to the second then returns True, otherwise False.

Program:

a = 20
b = 10

print(a == b)
print(a != b)
print(a > b)
print(a < b)
print(a >= b)
print(a <= b)

Output:

False
True
True
False
True
False

Logical Operators

Logical operators are used to combine multiple boolean expressions.

  • Logical AND(and) : This operator returns True only if both conditions are True. If either condition is False, then result is False.
  • Logical OR(or): This operator returns True if at least one condition is True. It returns False only when both conditions are False.
  • Logical NOT(not): It reverses the logical value. If a condition is True, then it becomes False, and if a condition is False then it becomes True.

Program:

a = True
b = False

print(a and b)
print(a or b)
print(not a)

Output:

False
True
False

Assignment Operators

Assignment operators are used to assign values to variables.

  • Simple assignment(=) : It is used to assign a value to a variable.
  • Add and assign(+=) : It adds the right-hand value to the variable and assigns the result back.
  • Subtract and assign(-=) : It subtracts the right-hand value from the variable and assigns the result back.
  • Multiply and assign(*=) : It multiplies the variable by the right-hand value and assigns the result back.
  • Divide and assign(/=) : It divides the variable by the right-hand value and assigns the result back.
  • Modulo and assign(%=) : It stores the remainder of division back into the variable

Program:

a = 10

a += 5
print("After += 5: " + str(a))

a -= 5
print("After -= 5: " + str(a))

a *= 5
print("After *= 5: " + str(a))

a /= 5
print("After /= 5: " + str(a))

a %= 5
print("After %= 5: " + str(a))

Output:

After += 5: 15
After -= 5: 10
After *= 5: 50
After /= 5: 10.0
After %= 5: 0.0
