File Operations in Python - III

File Operations in Python allow programs to read, write, append, and manage data stored in files. File handling is an essential part of programming as it enables data persistence, meaning information can be stored permanently and accessed later. Python provides built-in functions and methods to perform file operations efficiently and securely. File handling helps in organizing data externally, reducing memory usage, and managing large datasets effectively. By using file operations, real-world applications such as data logging, report generation, database management, and configuration storage can be implemented easily. In this lab, you will study different file handling techniques in Python, including opening files in different modes (read, write, append), using the with statement, reading files line by line, writing structured data, checking file existence, and performing basic file manipulation tasks.

File Operations in Python

File handling in Python is used to create, read, write, append and manage files. It allows data to be stored permanently instead of keeping it only in memory. 

open(filename, mode)

File Opening Modes

r  → Read file
w  → Write file
a  → Append data
x  → Create file
rb → Read binary file
wb → Write binary file

Create and Write File

file = open("sample.txt","w")

file.write("Hello Python")

file.close()

Read File

file = open("sample.txt","r")

print(file.read())

file.close()

Read Line by Line

file = open("sample.txt","r")

for line in file:
    print(line)

file.close()

Append Data into File

file = open("sample.txt","a")

file.write("\nNew Data")

file.close()

Using with Statement

with open("sample.txt","r") as file:

    print(file.read())

File Methods

read()      → Read complete file

readline()  → Read one line

readlines() → Read all lines

write()     → Write data

close()     → Close file

Check File Exists

import os

if os.path.exists("sample.txt"):
    print("File Exists")
else:
    print("File Not Found")
